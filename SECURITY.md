# Security Policy

## Supported versions

This project is pre-1.0 and under active development. Only the latest commit on `main` is supported. There are no
backported security releases.

## Reporting a vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Email [security@temporal.io](mailto:security@temporal.io) with a description of the issue, steps to reproduce, and any
relevant details. We will acknowledge your report within five business days and work with you on a fix and disclosure
timeline.
